## 🚀 Projetos em Destaque

### [teamcubation-partidas-de-futebol](https://github.com/yuricapella/teamcubation-partidas-de-futebol)

> API REST modular para gestão de partidas, clubes, estádios, rankings e retrospectos, com arquitetura multi-módulo, integração RabbitMQ e jobs agendados.  
**Tecnologias:** Java 21, Spring Boot 3.5.3, Spring Web, Data JPA, Validation, Springdoc OpenAPI, RabbitMQ, ModelMapper, Maven multi-módulo, JUnit 5, Mockito.  
✨ *Destaques:*  
- Buscas complexas, filtros avançados (goleada, mandante/visitante), Strategy para ranking
- Mensageria com producer/consumer RabbitMQ e DLQ
- Integração externa ViaCep para endereços de estádios
- Swagger OpenAPI completo (com exemplos de erro e response)
- Jobs agendados configuráveis via cron para cálculos automáticos de ranking
- Documentação modular e extensa
